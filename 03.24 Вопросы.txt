Для экономии времени предусмотрена сокращённая запись арифметических операторов.

x += y;    // То же самое, что и x = x + y;
x -= y;    // То же самое, что и x = x - y;
x *= y;    // То же самое, что и x = x * y;
x /= y;    // То же самое, что и x = x / y;
x %= y;    // То же самое, что и x = x % y;

==================Ответ================
Когда переопределяешь значение переменной используя её старое значение, то можно применить короткую запись.

int kmLeft = 1000;  						// декларирую новую переменную и определяю её значение.
... 		 								// допустим программа делают свою работу

if(distanceTraveled > 0){					// Предполагаемая программа дошла до места где ей необходимо уменьшить оставшуюся дистанцию (kmLeft) на то, которое прошли (distanceTraveled)
	kmLeft = kmLeft - distanceTraveled; 	// что бы назначить новое значение (оператор "=") в переменную обозначенную с лево, мне нужно узнать результат вычислений с право:
//			   1	3     2					//  1 - узнаю значение переменной kmLeft 1000. Так как в предполагаемой программе эта переменная не переназначалась то значение осталось изначальное
											//  2 - узнаю значение переменной distanceTraveled - допустим 125
											//  3 - результат вычислений - 875 записывается как новое значение переменной
											
	kmLeft -= distanceTraveled;				// короткая форма записи позволяет немного локаничней записать, но под капотом делается те же операции в том же порядке.
